Open Researcher and Contributor ID [ORCID (Open Researcher and Contributor ID)]

Last updated on Feb 26, 2026
Also available in: Arabic | English
 

Definition: Eine Organisation, die ein Register mit dauerhaften eindeutigen Identifikatoren (ORCID iDs) für Forscher:innen und Akademiker:innen bereitstellt, so dass diese Nutzer:innen ihre digitalen Forschungsdokumente und andere Beiträge mit ihrem ORCID-Eintrag verknüpfen können. Die Registrierung für eine ORCID iD ist kostenlos aufhttps://orcid.org/register.

Verwandte Begriffe: Authorship, DOI (digital object identifier), Name Ambiguity Problem

Referenzen:

  • Haak, L. L., Fenner, M., Paglione, L., Pentz, E., & Ratner, H. (2012). ORCID: A system to uniquely identify researchers. Learned Publishing, 25(4), 259–264. https://doi.org/10.1087/20120404
  • ORCID. (n.d.). ORCID. https://orcid.org/
